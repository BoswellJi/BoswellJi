## 概述

* webgl有两条平行的绘制流水线：几何绘制流水线和像素流水线；

* 纹理映射作为片元处理的一部分执行；
* 可以把纹理应映射视为明暗着色处理的一部分；
* 但是这部分的处理过程是基于片元逐级进行的；
* 纹理坐标与法向量和颜色的处理方式非常类似，它们都是作为webgl的状态与顶点相关联的，然后在多边形的表面对顶点的纹理坐标进行插值计算；
