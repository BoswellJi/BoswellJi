# 算法

* 算法是一个比计算机程序更高的抽象；
* 精确地定义一个操作序列的规则集；
* 算法就是如何使用数据结构的方法；

## 算法复杂度

* 使用`步数`来描述运行时间。`一步`就是计算的基本单位。通过测试`计算从开始到结束共执行多少步`来求得算法的运行时间；
* O这个符号的意思是：忽略重要项以外的内容；

## 时间复杂度

* 描述算法运行时间的函数；O(n): 运行时间根据输入n产生的变化程度；
