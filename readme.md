## 学习方法

1. 每天坚持学：English，JavaScript，WebGL；
2. 带着学：uniapp, vue,数据结构和算法， 设计模式；
3. 一定不能让自己陷入，重复学习，反复学习，那些非重点的部分的知识，这会干扰判断；
