# js 模块打包器(模块打包器)

## 特点

* 原生只支持es module，在不支持es module的环境中会智能降级；（想要导入其他模块类型的模块代码需要使用插件）
* 根据es module的静态分析，可以对代码进行树摇，将废弃代码处理掉；
* 生成的代码不会带有运行时，webpack带有自己的模块加载库运行时；

## tree-shaking的几种代码

* 代码不会被执行到
* 代码只影响死变量
* 代码执行的结果不会被用到
* 尽量不要写带有副作用的代码；
  - 副作用：改变环境中的变量的代码；
