# Nitro Presentation

A Slidev presentation about Nitro, the high-performance full-stack framework.

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Export to PDF
pnpm export
```

## Presentation Content

- Introduction to Nitro
- Nitro vs H3
- Architecture Overview
- Core Features
- API Routes
- Middleware
- Deployment Targets
- Performance Features
- Integration with Nuxt
