## 种类

* 关系代词

有连词功能，引导定语从句；

who,whom（宾格
which
that

whose: 所有格

1. 关系代词前要有先行词（表人或事物的名词）
2. 关系代词在所引导的定语从句中要作主语、宾语和表语；否则关系单词前一定要有介词

* whom在句中既不能作主语也不能做宾语。因此要加介词;
* 介词置后，可以将关系代词省略；

which除了代替事物，还可以代替之前的句子，但要用逗号隔开

* 关系副词

where =in which,at which,on which
when = in which,at which,on which
why = for which
how = in which

* 复合关系代词

whoever,whomever
whichever
what whatever
