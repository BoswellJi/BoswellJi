## 重复

* 非贪婪的重复
```
注意：?号是放在重复模式字符后面的
/a+/.exec('aaa') => ['aaa']

/a+?/.exec('aaa') => ['a']
```

## 选择，分组，和引用
```
指定选择项

/a|ac/

子表达式分组
/(ab)(bc)/

引用前一子表达式, \1表达：第一个分组匹配到的文本内容
/(a)bc\1/ 'abca'
```

## 指定匹配位置

* `^`:

* `$`:

* `\b`: 一个单词边界

* `\B`: 一个非单词边界

* `(?=p)`: 零宽正向先行断言，要求接下来的字符都与p匹配

* `(?!p)`: 零宽负向先行断言,要求接下来的字符与p不匹配
