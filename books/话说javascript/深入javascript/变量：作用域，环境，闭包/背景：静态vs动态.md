你可以从两个角度测试程序的工作原理：

## 静态地或者词法地

检查程序是否存在源代码中，没有运行。给出下面的代码，我们能做出函数g被嵌套在函数f内部的断言：

``` javascript
function f() {
    function g() {}
}
```

形容词词法的和静态的用法相同，因为两者都属于程序的词典（文字，来源）。

## 动态地

你测试当执行程序的时候发生了什么（在运行时中），给出下面的代码：

``` javascript
function g() {}

function f() {
    g();
}
```

当我们调用f()函数，它会调用g()函数，运行时期间，g一直被f调用，表示一个动态的关系。
